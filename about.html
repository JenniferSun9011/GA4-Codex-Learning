About this project

GA4 and Codex ecommerce data lab

AI Ecommerce Lab is a static demo store created for learning GA4 ecommerce tracking, conversion funnel analysis, and future Codex-assisted data diagnosis.

This is not a real store. It does not process real payments, collect real personal data, or fulfill orders.

What this site is for

  • Generate GA4 recommended ecommerce events.
  • Practice traffic, product, cart, checkout, and purchase analysis.
  • Build a safe foundation for future GA4 Data API reporting.
